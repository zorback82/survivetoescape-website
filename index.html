    C'est quoi ce mode de jeu ?

    Tout d'abord, ce mode de jeu se déroule sur Minecraft, il est développé pour la version 1.15.2 de Minecraft. Ce mode de jeu peut se jouer de plusieurs façons :

    • 4 équipes de 3 joueurs (12 joueurs au total)
    • 3 équipes de 3 joueurs (9 joueurs au total)
    • 4 équipes de 2 joueurs (8 joueurs au total)
    • 3 équipes de 2 joueurs (6 joueurs au total)

    Le but de ce mode de jeu est d'être la première équipe à s'échapper du monde en débloquant la clé finale qui elle-même ouvre le portail final. Vous pouvez trouver plus d'informations en cliquant sur le bouton ci-dessous pour accéder aux explications.

    A propos du plugin

    Survive To Escape est développé sous forme de plugin pour Minecraft Java.

    Il n'est pour le moment pas disponible, il est uniquement privé. Aucune date de sortie n'est prévue pour le moment.
