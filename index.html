C'est quoi ce mode de jeu ?

Tout d'abord, ce mode de jeu se déroule sur Minecraft, il est développé à la base pour la version 1.15.2 de Minecraft (c'est donc la version recommandée). Ce mode de jeu peut se jouer de plusieurs façons :

  • 4 équipes de 3 joueurs (12 joueurs au total)
  • 3 équipes de 3 joueurs (9 joueurs au total)
  • 4 équipes de 2 joueurs (8 joueurs au total)
  • 3 équipes de 2 joueurs (6 joueurs au total)

Le but simple de ce mode de jeu est d'être la première équipe à s'échapper du monde en débloquant la clé finale qui elle-même ouvre le portail final. Vous pouvez trouver plus d'informations en cliquant sur le bouton ci-dessous pour accéder aux explications.

Concernant le plugin Minecraft :

Survive To Escape est développé sous forme de plugin pour Minecraft Java.

Le plugin est entièrement développé par ZorBack. Concernant le développement du mode de jeu, Plash Har aide aussi pour le côté Front-End (côté jeu, directement sur Minecraft).

Le plugin n'est pas encore sorti en public, il est encore en version Alpha, mais vous pouvez continuer à suivre son développement sur ce site web.

Est-ce que je peux participer à une édition de Suvive To Escape ?

Pour le moment, il est important de noter que les participants aux éditions de Survive To Escape en version Alpha sont uniquement des personnes proches ou qui connaissent le staff.

Si vous souhaitez participer aux éditions dans le futur, vous pouvez rejoindre le serveur Discord et contacter en message privé les organisateurs des évènements : Moustic (organisateur principal) ou ZorBack (organisateur et développeur).
